# **App Name**: DevCard

## Core Features:

- User Registration: Allow users to register with a login, password, and display name. Password will be stored in hashed format.
- User Authentication: Authenticate users using JWT or session-based authentication. User login is for technical use and not displayed publicly.
- Personal CV Landing Page: Showcase developer's name, short description, technology stack, and links to GitHub/LinkedIn.
- Project Showcase: Display a grid of project cards, each containing the project name (from repository name), a short description, and links to comments and repository.
- Project Comments: Enable authorized users to add comments to projects. Comments display the user's display name, not the login.
- Checklist Display: Display a grid of projects, each with a checklist of tasks.
- Checklist Management: Allow admin users to add, delete, and edit tasks in the checklist. Track the completion status of each task in the database.

## Style Guidelines:

- Primary color: Navy blue (#3B5998) to convey professionalism and stability.
- Background color: Light gray (#F0F2F5), offering a clean and modern backdrop.
- Accent color: Teal (#008080) to draw attention to key elements, such as buttons and links, and to create contrast. 
- Headline font: 'Space Grotesk', a modern sans-serif, to provide a techy, clean feel for titles.
- Body font: 'Inter', a grotesque-style sans-serif for readability and modern appeal; pairs well with Space Grotesk. 
- Simple, line-based icons to maintain a minimalist aesthetic.
- Card-based layout with clear, consistent spacing to provide structure and readability.
- Subtle animations on hover to give interactive feedback.
